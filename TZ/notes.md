# Техническое задание: Система заметок

### 1. Логика структуры и вложенности
* **Беспапочная система:** Папок не существует. Каждая запись — это страница, которая может содержать внутри себя другие страницы.
* **Бесконечная вложенность:** Возможность создавать страницы внутри страниц по принципу «матрешки». Глубина уровней не ограничена.
* **Функция Toggle (Сворачивание):** Если у заметки есть вложенные уровни, рядом с ней появляется стрелка, позволяющая скрыть или раскрыть всё внутреннее содержимое.
* **Навигация (Breadcrumbs):** В верхней части страницы отображается путь из заголовков (например: *Работа > Проекты > Дизайн*), позволяющий в один клик вернуться на любой уровень назад.

---

### 2. Инструментарий работы с текстом
* **Верхняя панель управления:** Фиксированное меню в шапке (над заголовком), содержащее:
    * **Шрифт:** Жирный, курсив, подчеркнутый, зачеркнутый.
    * **Заголовки:** Кнопки переключения уровней текста (H1, H2, H3).
    * **Списки:** Создание маркированных и нумерованных списков, а также горизонтальный разделитель контента.
    * **Цвет:** Инструменты для смены цвета текста и заливки фона («маркер»).
* **Редактируемый заголовок:** Поле для названия заметки («Untitled Note») отделено от основного текста тонкой горизонтальной линией.
* **Чистое поле:** Текст начинается сразу под заголовком без лишних отступов и рамок.

---

### 3. Организация пространства (Боковая панель)
* **Список страниц:** Панель со списком всех созданных заметок.
* **Управление списком:** Кнопки для создания новой страницы (иконка документа) и кнопка «Trash» для удаления/корзины.
* **Карточка заметки:** В списке отображается название заметки. При клике на неё содержимое открывается в центральной области.
